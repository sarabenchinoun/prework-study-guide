HTML

  • HTML stands for Hypertext Markup Language
  • HTML elements build the structure of the webpage in a way that is easy to navigate.
  • A !DOCTYPE is required at the top of any html code file to tell the computer what type of file it is.
  • After the DOCTYPE it is necessarcy to add an open and close html tag and to write all the html code within this tag.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Common element types used to add content to pages are text, image, header and list elements.
  • Common element types that are used to divide and describe a page are meta-related, content division and semantic elements.

CSS

  • CSS stands for Cascading Style Sheet
  • CSS determines the styles of a webpage in order to make a website visually appealing and simpler to use because of the polished design and consistent organization.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • All HTML elements are boxes made up of four components: a content container, padding, border, and margin.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS style files need to be connected to the index with code in the head.

Git

  • Github is an online site used to store and hold repositories.
  • Github allows developers to work and collaborate on the same projects simultaneously.
  • Developers have the ability to take the diffrent branches and merge them together in Github.
  • Pull requests enables a developer to check all the new code to make sure it all flows, and other developers to review his work for him and give him feedback where needed.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript can add interactivity to the website
  • JavaScript is an object oriented code
  • A variable is a named container that allows us to store data in our code.
  • There are seven fundamental data types in JavaScript: string, number, boolean, null, objects, undefined, and symbol
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements and loops can be used to control the order in which code is executed, or if it is even executed at all.
  • A function is simply a set of instructions that tells the computer how to perform a certain task, and it can be reused in different places.
